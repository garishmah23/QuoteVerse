# 📖 QuoteVerse - Modern Random Quote Generator

A modern and elegant desktop application built with Python and Tkinter that displays inspirational quotes with a beautiful dark-themed user interface.

QuoteVerse fetches quotes from an online API and automatically falls back to local quotes when the internet is unavailable.

---

## ✨ Features

* 🎲 Random Quote Generator
* 🌐 Online Quote API Integration
* 💾 Save Favorite Quotes
* 📋 Copy Quotes to Clipboard
* 🎨 Modern Dark-Themed UI
* 🔄 Automatic Offline Backup Quotes
* 🏗️ Object-Oriented Programming (OOP)
* 🖥️ Desktop Application using Tkinter

```bash

```

---

## 🎯 How It Works

1. Application starts.
2. A random quote is displayed.
3. User can:

   * Generate a new quote
   * Save quote to favorites
   * Copy quote to clipboard
4. If API is unavailable, local quotes are used automatically.

## 📁 Data Files

### quotes.json

Stores local backup quotes.

### favorites.json

Stores user-saved favorite quotes.
